This package is used for I/O and preprocessing of data from the [NASA Wind Satellite](https://wind.nasa.gov/)'s radio instrument WAVES, with a focus on Auroral Kilometric Radiation.

**License:** CC0-1.0

**Support:** please [create an issue](https://github.com/arfogg/wind_waves_pipeline/issues) or contact [arfogg](https://github.com/arfogg) directly. Any input on the code / issues found are greatly appreciated and will help to improve the software.
